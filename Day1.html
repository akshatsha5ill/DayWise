Cheat Sheet

The Four Shapes You Need

Every flowchart is built from these four building blocks. Refer back as you solve the exercises below.

Terminal

Oval

Marks where a process begins and ends. Every flowchart needs exactly one of each — no more, no fewer.

Process

Rectangle

Represents an action or operation. "Add two numbers," "Save file," "Send email" — each gets its own box.

Decision

Diamond

The brain of the flowchart. Splits the path into exactly two branches — always a strict yes or no, never maybe.

I/O

Parallelogram

Represents data entering or leaving the system. "Read name," "Print result" — always a parallelogram, never a rectangle.

Q i
Sum of Two Numbers Easy

Hint: You need ovals (start/end), parallelograms (input/output), and a rectangle (calculate) — no decisions.

Logic Steps

  1. Oval Start
  2. Para Input first number (A)
  3. Para Input second number (B)
  4. Rect Calculate Sum = A + B
  5. Para Output Sum
  6. Oval End

Q ii
Simple Interest Easy

Hint: The formula is SI = (P × R × T) / 100. Three parallelogram inputs, one rectangle formula, one parallelogram output.

Logic Steps

  1. Oval Start
  2. Para Input Principal (P)
  3. Para Input Rate (R)
  4. Para Input Time (T)
  5. Rect Calculate SI = (P × R × T) / 100
  6. Para Output SI
  7. Oval End

Q iii
Maximum of Three Numbers Medium

Hint: Compare two numbers first, then compare the winner with the third. You'll need two diamonds.

Logic Steps

  1. Oval Start
  2. Para Input A, B, C
  3. Diamond Is A > B?

    → Yes path:

    1. Diamond Is A > C?

      → Yes: Max = A

      → No: Max = C

    → No path:

    1. Diamond Is B > C?

      → Yes: Max = B

      → No: Max = C

  4. Para Output Max
  5. Oval End

Flowchart

Q iv
Number is Prime or Not Hard

Hint: A prime number is only divisible by 1 and itself. You'll need a loop that checks divisibility from 2 up to N−1, and a counter or flag that flips if any divisor is found. Think: diamond inside a loop.

Logic Steps

  1. Oval Start
  2. Para Input N
  3. Rect Set I = 2, Flag = 0
  4. Diamond Is I ≤ N − 1?

    → Yes (keep checking):

    1. Diamond Is N % I == 0?

      → Yes: Set Flag = 1 (not prime)

      → No: do nothing

    2. Rect Increment I = I + 1
    3. Loop Go back to "Is I ≤ N − 1?"

    → No (loop done):

    1. Diamond Is Flag == 0?

      → Yes: Print "Prime"

      → No: Print "Not Prime"

  5. Oval End

Flowchart

Q v
Area of a Circle Easy

Hint: Formula is A = π × r². One parallelogram input, one rectangle calculation, one parallelogram output. No decisions needed.

Logic Steps

  1. Oval Start
  2. Para Input radius (r)
  3. Rect Calculate Area = 3.14159 × r × r
  4. Para Output Area
  5. Oval End
